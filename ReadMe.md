# eCommerce Website

## Overview                
- Technical: JSP (Jakarta Server Pages) and Servlet
- Relational database management system : **MySQL**
- Design pattern : **Model - View - Controller (MVC)**
- Integrated development environment (IDE) : **Jetbrains Intellij**
